# NetDiag

Асинхронная утилита на базе Python3 для диагностики, автоопределения, регистрации и активного управления абонентскими терминалами (ONU) на OLT-коммутаторах ZTE C300 и ZTE C600.

## Ключевые возможности
- **Параллельный опрос сети OLT**: Высокоскоростной одновременный сбор списка незарегистрированных (unconfigured) ONU по всему пулу OLT коммутаторов с использованием асинхронного ввода-вывода (`asyncio`).
- **Автоопределение положения ONU (Auto-Discovery)**: Поиск OLT-коммутатора и конкретного физического PON-порта, на котором зарегистрирован или ожидает регистрации абонент, по его серийному номеру.
- **Глубокая диагностика**: Сбор уровней PON сигналов (Tx/Rx), линков Ethernet-порта, прокачки трафика, IP/MAC-адресов абонентского оборудования и истории аварий за одну сессию.
- **Интерактивная регистрация (`--reg`)**: Поиск неконфигурированной ONU и её автоматическая регистрация на OLT C600 и C300. Поддерживается автоматический откат к совместимому профилю для разных версий прошивок, а также специфическая конфигурация для аппарата `172.31.2.13` (2.13) под нужды конкретных моделей (ZTE-F601, ручные vport и relay-агенты).
- **Удаленное управление**: 
  - Удаленная OMCI-перезагрузка ONU (`--reboot`).
  - Полная деконфигурация (удаление) ONU с интерфейса OLT (`--remove`).
- **Автоматическая аналитика**: Оценка состояния оптической линии и абонентского порта на базе встроенных правил с формированием лаконичных рекомендаций для технической поддержки.
- **Полная изоляция кода**: Платформы ZTE C300 и ZTE C600 обслуживаются изолированными друг от друга драйверами и парсерами для безопасного обслуживания в условиях разных прошивок.

## Системные требования
- Python 3.8+
- Зависимости отсутствуют (утилита построена исключительно на стандартной библиотеке Python).

## Быстрый старт
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ZephronixQ/NetDiag.git
   cd NetDiag
   ```
2. Создайте файл `config.py` и заполните массив `OLT_DEVICES` параметрами ваших коммутаторов. 

## Использование

### 1. Вывод справки
```bash
python main.py --help
```

### 2. Поиск незарегистрированных ONU
Параллельный опрос всех OLT, указанных в конфигурационном файле:
```bash
python main.py --uncfg
```

### 3. Запуск диагностики абонента
Скрипт автоматически найдет нужный OLT и соберет аналитику по серийному номеру:
```bash
python main.py --gpon XPON57000562
```

### 4. Удаленная перезагрузка ONU
```bash
python main.py --gpon XPON57000562 --reboot
```

### 5. Удаление конфигурации ONU с порта OLT
```bash
python main.py --gpon XPON57000562 --remove
```

### 6. Регистрация новой ONU
Программа найдет незарегистрированное устройство с указанным серийным номером на нужном порту и выполнит последовательную регистрацию с автоматическим сохранением (`write`):
```bash
python main.py --reg XPON57000562 --vlan 1916 --int 1/3/8 --port 55 --id 10022870
```
