# Changelog

Все важные изменения этого проекта будут документироваться в данном файле.

## [1.0.0] - 2026-02-15
### Добавлено
- Первая стабильная версия (Initial Release).
- Модульная архитектура с разделением логики под платформы ZTE C300 и ZTE C600.
- Функция параллельного опроса сети OLT для быстрого поиска незарегистрированных ONU (`--uncfg`).
- Функция автоопределения расположения ONU по серийному номеру при запуске диагностики (`--gpon`).
- Модуль автоматической аналитики показателей затухания, линков Ethernet-порта и логов аварий.
