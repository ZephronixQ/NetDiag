# Changelog

Все важные изменения этого проекта будут документироваться в данном файле.

## [1.1.0] - 2026-07-06
### Добавлено
- Функционал активного управления ONU: удаленная OMCI-перезагрузка (`--reboot`) и удаление конфигурации с OLT порта (`--remove`).
- Мастер автоматической регистрации ONU (`--reg`) с поддержкой динамического определения нужного коммутатора по unconfigured-списку.
- Автоматическая обработка синтаксиса на OLT C300: резервный откат конфигурации при отсутствии поддержки опции `remote VLAN` на старых прошивках.
- Специфический сценарий регистрации для OLT `192.168.2.13` (2.13): интеграция модели ZTE-F601, кастомных vport, лимитов, dhcpv4-relay и корректного извлечения RID через `show port-identification`.
- Функция автоматического сохранения конфигурации (`write`) на OLT после завершения сессии регистрации.

### Исправлено
- Исправлен парсер неконфигурированных ONU на платформе ZTE C600, который ранее игнорировал строки, содержащие названия моделей (например, `MTE-X100DG`) между портом и серийным номером.

## [1.0.0] - 2026-07-05
### Добавлено
- Первая стабильная версия (Initial Release).
- Модульная архитектура с разделением логики под платформы ZTE C300 и ZTE C600.
- Функция параллельного опроса сети OLT для быстрого поиска незарегистрированных ONU (`--uncfg`).
- Функция автоопределения расположения ONU по серийному номеру при запуске диагностики (`--gpon`).
- Модуль автоматической аналитики показателей затухания, линков Ethernet-порта и логов аварий.
