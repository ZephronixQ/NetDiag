# Roadmap развития проекта

Основные возможности мониторинга и активного конфигурирования GPON-сетей и коммутаторов доступа IPoE успешно интегрированы в CLI-интерфейс.

## Выполнено в релизах 1.0.0 - 1.2.0
- [x] **Параллельный опрос сети GPON (`--uncfg`)**: Высокоскоростной парсинг неконфигурированных ONU по всем коммутаторам.
- [x] **Автоопределение ONU (`--gpon`)**: Динамический поиск OLT по серийному номеру абонента.
- [x] **Удаленная перезагрузка ONU (`--reboot` GPON)**: Принудительный перезапуск ONU с обработкой подтверждений со стороны OLT.
- [x] **Удаление ONU (`--remove`)**: Полная деконфигурация ONU с порта.
- [x] **Регистрация ONU (`--reg`)**: Авторегистрация на C600/C300 с функцией автоматического сохранения настроек.
- [x] **Диагностика IPoE-абонентов (`--ipoe`)**: Сбор метрик, маков, DHCP и логов L2-коммутаторов доступа ZTE.
- [x] **Снятие блокировок MAC Protect на коммутаторах**: Разблокировка портов в режиме Restrict и сброс статистики ошибок одной командой.
