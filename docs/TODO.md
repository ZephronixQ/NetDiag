# Roadmap развития проекта

В будущих релизах планируется расширение возможностей CLI-интерфейса и перевод утилиты из режима мониторинга в режим активного управления ONU:

- [ ] **Интерактивная регистрация (`--register <SERIAL_NUMBER>`)**:
  Автоматический подбор свободного индекса на порту, применение стандартных шаблонов конфигурации (Line Profile, Service Profile) и сохранение настроек на OLT.
  
- [ ] **Быстрое удаление (`--delete <SERIAL_NUMBER>`)**:
  Команда полной деконфигурации абонентского терминала с порта OLT по его серийному номеру.
  
- [ ] **Удаленная перезагрузка (`--reboot <SERIAL_NUMBER>`)**:
  Команда на принудительную перезагрузку абонентской ONU через OMCI-запрос OLT.
