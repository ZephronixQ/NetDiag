# Roadmap развития проекта

Основные возможности мониторинга и активного конфигурирования были успешно интегрированы в CLI-интерфейс.

## Выполнено в релизах 1.0.0 - 1.1.0
- [x] **Параллельный опрос сети (`--uncfg`)**: Высокоскоростной парсинг неконфигурированных ONU по всем коммутаторам.
- [x] **Автоопределение ONU (`--gpon`)**: Динамический поиск OLT по серийному номеру абонента.
- [x] **Удаленная перезагрузка (`--reboot`)**: Принудительная перезагрузка ONU с обработкой подтверждений со стороны OLT.
- [x] **Удаление ONU (`--remove`)**: Полная деконфигурация ONU с порта.
- [x] **Регистрация ONU (`--reg`)**: Шаблонизатор конфигурации для C600 и C300 (включая ветвление под версию прошивки C300 и особенности OLT 2.13) с функцией автоматического сохранения настроек.
